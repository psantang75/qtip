Local Setup (QTIP)
1) Prerequisites
Node.js 18+ (npm 9+)
MySQL 8+
Git
2) Clone and open project
git clone <your-repo-url>
cd TypeScript_Projects/qtip
3) Install dependencies
From repo root:

npm install
(Equivalent workspace command also exists: npm run install:all)

4) Configure backend environment
The backend reads env from backend/.env.

Use existing backend/.env as a template, but update DB/JWT values for your machine.
At minimum, make sure these are valid for your local/dev setup:
NODE_ENV=development
PORT=3000
DB_HOST
DB_USER
DB_PASSWORD
DB_NAME
JWT_SECRET
REFRESH_TOKEN_SECRET
ALLOWED_ORIGINS=http://localhost:5173,http://localhost:3000
5) Prepare database
Create DB in MySQL:

CREATE DATABASE qtip;
Then import schema/data.

Important note for this repo: I only found scripts/build_production_database.sql, which is a cleanup/truncate script for an existing DB, not a full schema bootstrap script.
So for a fresh local DB, you’ll need a schema/dump from your team/environment and import it, e.g.:

mysql -u <user> -p qtip < <schema_or_dump>.sql
6) Run the app
From root:

npm run dev
This starts:

Backend on http://localhost:3000
Frontend on http://localhost:5173
7) Verify
Frontend: http://localhost:5173
Backend health: http://localhost:3000/health
API docs: http://localhost:3000/api-docs
